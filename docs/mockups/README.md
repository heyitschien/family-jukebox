# Cousin Radio Mockups

This folder contains static HTML mockups for exploring UI directions before implementation.

## Files

- [`cousin-radio-mobile-first-landing.html`](./cousin-radio-mobile-first-landing.html) — mobile-first responsive landing page mock for the next-stage private-first / public-radio Cousin Radio experience.

## How to use

Open the HTML file directly in a browser, or copy the structure into the Next.js app as implementation reference.

The mockup is intentionally self-contained:

- no build step
- no external dependencies
- placeholder icons and CSS-generated art
- responsive layout for mobile and desktop

## Design direction

```text
A beautiful public front door
+
A private family music home
+
A joyful public radio window
```

Core product promise:

```text
Your family's songs start private. You choose what to share.
```
