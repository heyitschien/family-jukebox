# Presentations

This folder contains discussion decks and meeting materials for Cousin Radio / Family Jukebox.

## Decks

- [`cousin-radio-ai-possibility-10-slide-deck.md`](./cousin-radio-ai-possibility-10-slide-deck.md) — accessible 10-slide family deck explaining AI, Cousin Radio, personal creation, family music, and the shift from consuming culture to creating family culture.

## Tone

These decks should be:

- warm
- clear
- non-technical when possible
- invitational rather than forceful
- grounded in real family use
- focused on possibility, love, memory, and joy
